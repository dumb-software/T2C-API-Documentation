## Pegase - param

Obtenir la dernière date de la synchronisation avec l'appli?

### Obtenir la date

#### URL
```
  GET http://pegase.t2c.fr/Param.txt
```


---

### Exemple de réponse:

```
20250606190353239
```

La réponse est sous format `YYYYMMDDHHMMSSSSS`

Exemple:

    * YYYY = Année (2025)
    * MM = Mois (06)
    * DD = Jour (06)
    * HH = Heure (19)
    * MM = Minutes (03)
    * SS = Secondes (53)
    * SSS = Millisecondes (239)
